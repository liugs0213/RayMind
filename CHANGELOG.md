## [0.1.0] - 2024-01-XX

### 新增
- ScheduleMesh 核心类实现
- Broker 和 Agent 管理
- 三种调用方式：`.all()`, `.choose()`, `.shard()`
- 基础调度器实现
- 资源注册表
- 抢占控制器
- 分片管理器
- PlacementGroup 控制器
- 插件系统框架
- 基础测试用例
- 使用示例
- 项目文档

### 技术特性
- 基于 Ray 2.0+ 的分布式调度
- 支持 Python 3.8+
- 可插拔的调度策略
- 标签化资源管理
- 动态扩缩容支持
- 优先级抢占机制
- 异步 metrics 收集

### 已知问题
- 状态管理功能需要完善
- 监控系统需要增强
- 性能优化空间较大
- 文档需要补充更多示例
